# 🚀 FinSight AI

AI-Powered Financial Intelligence Dashboard for Smarter Investment Decisions

🌐 Live Demo:

https://finsight-ai-np5kccgeogf4ru3hsukvpa.streamlit.app/

💡 Overview

FinSight AI is an end-to-end Machine Learning-powered financial analytics platform that transforms raw financial data into actionable insights.

Built with a focus on real-world financial use cases, this project integrates predictive modeling, NLP, and data analysis into a single interactive dashboard.

🎯 Key Highlights :
✅ Built a multi-module ML system combining prediction, NLP, and anomaly detection
✅ Implemented XGBoost-based stock forecasting model
✅ Applied FinBERT for financial sentiment analysis on real-world data
✅ Designed an interactive Streamlit dashboard for end-user accessibility
✅ Integrated multiple financial tools into a single unified platform
✅ Focused on practical business use cases (investment insights, risk detection, portfolio analysis)

🔥 Core Features

📈 Stock Price Prediction
Machine Learning model using XGBoost
Identifies trends and forecasts stock movements

📰 News Sentiment Analysis
NLP pipeline using FinBERT (HuggingFace Transformers)
Classifies news into positive, negative, or neutral sentiment

🧠 CEO Transcript Analysis
Extracts insights from earnings calls
Helps evaluate company outlook and leadership tone

🚨 Anomaly Detection
Detects unusual financial patterns
Useful for fraud detection and risk monitoring

💼 Portfolio Insights
Performance tracking and optimization suggestions

⚖️ Company Comparison
Compare firms using key financial indicators

🤖 AI Financial Advisor
Interactive assistant for financial queries
Provides intelligent, data-backed suggestions

📄 Report Generator
Generates structured financial reports
Useful for presentations and decision-making
