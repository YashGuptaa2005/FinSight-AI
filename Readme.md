# 🚀 FinSight AI

FinSight AI is a multi-page financial intelligence dashboard built using Machine Learning and Streamlit.

## 🔥 Features

- 📈 Stock Prediction (XGBoost)
- 📰 News Sentiment Analysis (FinBERT)
- 🧠 CEO Transcript Analysis
- 🚨 Anomaly Detection
- 💼 Portfolio Insights
- ⚖️ Company Comparison
- 🤖 AI Financial Advisor
- 📄 Report Generator

## 🛠️ Tech Stack

- Python
- Streamlit
- XGBoost
- HuggingFace Transformers (FinBERT)
- Pandas

## ▶️ Run Locally

```bash
venv\Scripts\activate
streamlit run app.py
